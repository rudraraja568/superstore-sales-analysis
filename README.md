#  Superstore Sales Analysis Dashboard
## Objective
To analyze sales, profit, and business performance using multiple tools and derive actionable insights.
## Tools Used
* SQL (MySQL)
* Microsoft Excel
* Tableau
* Power BI
## Dataset
Superstore dataset containing:
* Sales
* Profit
* Category
* Region
* Customer data
## Key Insights
* Technology category generated the highest sales (~4.7M)
* Overall profit margin is around 11.6%
* United States is the top-performing region
* Some sub-categories are loss-making
